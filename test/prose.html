n-prose 스타일 검증

이 문서는 @newtil/web의 n-prose 컴포넌트가 마크다운/HTML 본문을 일관되게 렌더링하는지 확인하는 페이지입니다. 모든 스타일은 @newtil/design-tokens의 변수를 참조합니다.

제목 계층

제목은 위·아래 여백을 두되, 위쪽 여백을 더 크게 주어 섹션 구분이 자연스럽도록 설계했습니다.

h3 레벨 제목

h4 레벨

소제목도 동일한 규칙.

h6 레벨 — 약한 강조용

텍스트 강조

굵은 글씨, 기울임, 밑줄, 취소선, 형광펜, 인라인 코드, 작은 글씨, Cmd + K로 검색을 엽니다.

링크

문서 내 링크는 secondary 색상으로 표시되며, hover 시 어두워집니다. 외부 링크도 동일하게 처리됩니다.
